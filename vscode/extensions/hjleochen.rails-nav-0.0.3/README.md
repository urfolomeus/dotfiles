## Rails navigation for Visual Studio Code

Navigation between related files.

![screenshot](https://raw.githubusercontent.com/hjleochen/vscode-rails-nav/master/rails-nav.png)

## Install
ext install rails-nav

## Default keybinding

* Alt + . (Alt + dot)
* Opt + . (Opt + dot Mac)
